# 什么是二进制
用 ^ 表示次方运算, 十进制的数位 (千位, 百位, 十位等) 全都是 10^n 的形式. 如 2871 = 2*10^3 + 8*10^2 + 7*10^1 + 1*10^0
二进制的数位则是 2^n 的形式. 如十进制的 53 等于二进制的 110101. 110101 = 1*2^5 + 1*2^4 + 0*2^3 + 1*2^2 + 0*2^1 + 1*2^0 = 32 + 16 + 0 + 4 + 0 + 1 = 53
